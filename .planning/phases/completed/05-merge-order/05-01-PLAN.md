    5. In `crates/smelt-core/src/git/cli.rs`, implement the new method for `GitCli`:
       ```rust
       async fn diff_name_only(&self, base_ref: &str, head_ref: &str) -> Result<Vec<String>> {
           let output = self.run(&["diff", "--name-only", base_ref, head_ref]).await?;
           if output.is_empty() {
               return Ok(Vec::new());
           }
           Ok(output.lines().map(|l| l.to_string()).collect())
       }
       ```

    6. Add a unit test in `crates/smelt-core/src/git/cli.rs` tests module:
