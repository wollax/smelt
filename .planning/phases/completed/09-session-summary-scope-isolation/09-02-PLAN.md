**1. Create `crates/smelt-core/src/summary/analysis.rs`** with:

```rust
//! Summary data collection — gathers per-session diff stats, commit messages,
//! and scope violations.
```

- `collect_summary()` async function:
  ```rust
  /// Collect a summary report for all completed sessions in a manifest.
  ///
  /// For each session, gathers:
  /// - Files changed with line counts (via `diff_numstat` + `diff_name_only`)
  /// - Commit messages (via `log_subjects`)
  /// - Scope violations (via `check_scope`)
  ///
  /// Sessions without a terminal `Completed` state in `session_states` are skipped.
  pub async fn collect_summary<G: GitOps>(
      git: &G,
      manifest: &Manifest,
      session_states: &HashMap<String, SessionRunState>,
      run_id: &str,
  ) -> crate::Result<SummaryReport>
  ```
  Implementation:
  1. For each session in the manifest where `session_states` shows `Completed`:
     a. Determine `base_ref`: use session's `base_ref` override if set, otherwise manifest's `base_ref`.
     b. Determine session branch name: `smelt/<manifest-name>/<session-name>` (same convention as WorktreeManager).
     c. Call `git.diff_numstat(base_ref, session_branch)` to get `(insertions, deletions, filename)` tuples.
     d. Call `git.diff_name_only(base_ref, session_branch)` to get complete file list (catches binary files that diff_numstat shows as `-\t-\tfile`).
     e. Call `git.log_subjects(&format!("{base_ref}..{session_branch}"))` to get commit messages.
     f. Merge diff_numstat and diff_name_only results:
        - Build a HashMap from diff_numstat: filename -> (insertions, deletions)
        - For files in diff_name_only but not in the numstat map, add with insertions=0, deletions=0 (binary files)
        - Result is `Vec<FileStat>`
     g. Call `check_scope(session_name, file_scope, shared_files, changed_file_paths)` to get violations.
     h. Build `SessionSummary` with computed totals.
  2. Compute `SummaryTotals` by summing across all session summaries.
  3. Build and return `SummaryReport`.

- `session_branch_name(manifest_name, session_name) -> String`:
  ```rust
  /// Derive session branch name from manifest and session names.
  /// Matches the convention used by WorktreeManager.
  fn session_branch_name(manifest_name: &str, session_name: &str) -> String {
      format!("smelt/{manifest_name}/{session_name}")
  }
  ```

- Handle errors gracefully: if `diff_numstat` or `diff_name_only` fails for a session (e.g., branch doesn't exist), log a warning via `tracing::warn!` and skip that session from the summary rather than failing the entire operation.

let summary_report = match crate::summary::collect_summary(
