<title>Create Rust workspace and crate skeletons</title>
<files>

<action>
1. Create workspace root `Cargo.toml` with:
   - `[workspace]` with `resolver = "2"` and `members = ["crates/*"]`
   - `[workspace.package]` with version "0.1.0", edition "2024", rust-version "1.85", license "MIT", repository "https://github.com/wollax/smelt"
   - `[workspace.dependencies]` with ALL dependencies from research (clap 4.5 with derive feature, tokio 1 with macros+rt-multi-thread+process, serde 1 with derive, toml 1, thiserror 2, anyhow 1, tracing 0.1, tracing-subscriber 0.3 with env-filter, console 0.16, which 8, assert_cmd 2, predicates 3, tempfile 3)

2. Create `crates/smelt-cli/Cargo.toml` per Example 2 in research:
   - Inherit version, edition, license, repository from workspace
   - `[[bin]]` named "smelt"
   - Dependencies: smelt-core (path), clap, tokio, anyhow, console, tracing, tracing-subscriber (all workspace)
   - Dev-dependencies: assert_cmd, predicates, tempfile (all workspace)

3. Create `crates/smelt-core/Cargo.toml`:
   - Inherit version, edition, license, repository from workspace
   - Dependencies: thiserror, serde, toml, tokio, tracing, which (all workspace)
   - Dev-dependencies: tempfile (workspace)

4. Create `crates/smelt-cli/src/main.rs` — minimal placeholder:
   ```rust
   fn main() {
       println!("smelt");
   }
   ```

5. Create `crates/smelt-core/src/lib.rs` — empty module with a doc comment:
   ```rust
   //! Smelt core library — git operations, orchestration logic, and domain types.
   ```
</action>
