2. **Missing integration test for dirty/force removal** — Only unit-tested. Severity: low.

### Phase 10: Real Agent Sessions
3. **Cancellation integration test** — Not implemented as CLI integration test. Unit-tested via `tokio::select!` structure; shares identical code path with timeout (which is integration-tested). Severity: cosmetic.
4. **`has_commits`/`steps_completed` not populated in orchestrator agent path** — AgentExecutor returns `has_commits: false` and `steps_completed: 0` because parsing Claude Code output for commit detection is deferred. Documented as v0.1.0 limitation. Severity: low.
5. **Incomplete lib.rs re-exports** — CLI imports types via `smelt_core::orchestrate::types::SessionRunState` instead of `smelt_core::SessionRunState`. Functional but not idiomatic. Severity: cosmetic.
